Este é um repositório feito para o trabalho de Inteligência Artificial com o seguinte tema: "A triagem automática de resíduos é um passo importante para otimizar a reciclagem. O seu objetivo é construir um classificador de imagens multiclasse para categorizar imagens de lixo em seis tipos: vidro, papel, papelão, plástico, metal e lixo orgânico."

Com base nisso, foram utilizados: 
- a base de dados do kaggle: https://www.kaggle.com/datasets/asdasdasasdas/garbage-classification
- o notebook do kaggle: https://www.kaggle.com/code/farnazmirfeizi/trash-garbage-type-detection-using-cnn

O trabalho então foi feito no notebook `trash-garbage-type-detection-using-cnn.ipynb`. Tenha noção que o notebook foi desenvolvido no ambiente do Google Colab, pois ele proporciona tanto um ambiente colaborativo pra toda equipe poder visualizar, além de possibilitar o uso de uma GPU (o uso limitado desse recurso foi o suficiente para o trabalho). 

O repositório possui os seguintes arquivos:
- `trash-garbage-type-detection-using-cnn.ipynb`: Notebook desenvolvido para treinamento do modelo CNN. Foi feito no Google Colab e baixado dele.
- `requirements.txt`: Um arquivo de dependências para rodar o notebook, em caso queira rodar localmente em vez de utilizar o Google Colab.
